# coc-code-runner

Code Runner for coc.nvim

## Install

```vim
:CocInstall coc-code-runner

```

## Configuration options

- `coc-code-runner.enabled`: Enable coc-code-runner extension, default: `true`
- `coc-code-runner.executorMap`: Set the executor of each language.

## Code Actions

- code runner: `coc-code-runner.run`

## Commands

- `coc-code-runner.run`: Run Code
